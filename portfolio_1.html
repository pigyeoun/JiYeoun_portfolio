    default 페이지 제작

    보다 쉬운 작업을 위해 메인, 상세페이지에서 사용가능한 반응형 레이아웃을 제작하여 프로그래밍 담당 팀원들에게 배포하였다. 메뉴에만 기능을 구현하였다.

    상세페이지 - 업사이클

    default 페이지 제작과 함께 맡게 된 페이지.
    함께 맡게 된 제품 페이지와 레이아웃이 비슷해 비교적 빠르게 작업하였다.

    상세페이지 - 제품

    default 페이지 제작과 함께 맡게 된 페이지.
    제품이 한 눈에 보이도록 바둑판형식의 레이아웃을 사용했다. 상단 메뉴에 링크를 이용해 제품과 그 제품에 대한 설명을 보기 쉽게 정리하였다.

    상세페이지 - 시공사례

    default 페이지 제작 후 맡게 된 페이지.
    이미지가 많아 반응형에서 제작 시간이 생각보다 많이 소모되었다. 별다른 설명을 덧붙이지 않고 심플하게 정렬했다.
